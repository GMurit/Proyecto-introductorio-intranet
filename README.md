# Proyecto Introductorio Intranet

Este proyecto es un ejercicio práctico para aprender a **conectar PHP con una base de datos MySQL** y realizar operaciones básicas de gestión de datos.  

Se trata de una **pequeña intranet** donde los profesores pueden iniciar sesión y gestionar cursos.

---

## Funcionalidades

- Inicio de sesión de profesores verificando la existencia en la base de datos.
- Acceso al **menú principal** tras iniciar sesión correctamente.
- Operaciones sobre cursos:
  - Consultar cursos existentes.
  - Modificar cursos (nombre del curso o nombre del docente).
  - Borrar cursos.
  - Agregar nuevos cursos.

El objetivo principal es **practicar la conexión a MySQL con PHP y la manipulación de datos** a través de consultas SQL.

## Requisitos

- PHP 7+  
- MySQL 5.7+ o 8  
- Servidor local (XAMPP, WAMP, MAMP, etc.)
